1)删除(裁剪拼接)
ary.splice(n,m) : 从索引n开始删除m个元素,把删除的内容以一个新数组的方式返回,原来数组发生改变
ary.splice(n) :
只传一个参数,表示从索引n开始,删除到数组的末尾,返回的是被删除项组成的新数组
ary.splice(0):
把原来数组中每一项都删除,(清空数组每一项),把之前每一项的值以一个新的数组返回(把之前的数组克隆了
一份一模一样的,这样的克隆会修改原来的数组),原有数组改变;
ary.splice():
数组中一项都没有删除,返回的是一个空数组,原有数组不发生改变;
2)修改(替换)
splice(n,m,x):
从索引n开始,删除m个,用x替换删除的部分,返回值:删除的内容以一个新的数组返回,原数组发生改变;
3)增加
splice(n,0,x):
从索引n开始,一个都不删除,返回的是一个空数组,把x增加到索引n的前面;关于数组的截取和查找(拼接)slice:实现数组的截取,在原来的数组中截取一部分;
语法:
1）slice(n,m)：从索引n开始找到索引为m处（不包括m），将找到的部分以一个新的数组返回，原来数
组不变；（m必须大于n）
ary.slice(n-1,m),找出第n项到第m项(包含第m项);
2）slice（n）：从索引n开始一直找到数组末尾,将找到的以一个新的数组返回,原来数组不变;
3）slice（0）或者slice（）：把原来的数组克隆一份一模一样的,以一个新数组返回,原来数组不变;concat把两个数组拼接到一起

->forEach:数组中有几项,我们的函数就执行几次,函数中的形参:item->当前遍历的数组中这一项的值;

->map和forEach的语法一样,但是比forEach多加了一个把原来内容进行替换的功能,原数组不变;

1.find 查找某个元素
返回true则将当前遍历的元素返回 找出数组中, 名字为jw的项,找到后立即返回不继续查找
var arr = [{name:’jw’},{name:’zfpx’},{name:’zry’},{name:’jw’}];
var obj = arr.find(function (item) {
if(item.name == ‘jw’){
return true;
}
});

console.log(obj); // jw

2.some 查看是否有满足条件的元素
返回true则结果为true 查看数组中, 名字是否有jw的项,找到后立即返回不继续查找

● var arr = [{name:’jw’},{name:’zfpx’},{name:’zry’},{name:’jw’}]; var flag = arr.some(function (item) { if(item.name == ‘jw’){ return true; } }); console.log(flag); // true

3.map
遍历每一项，用返回值替换当前项 将数组每个人的年龄累加10岁

var arr = [{name:’jw’,age:18},{name:’zfpx’,age:40},{name:’zry’,age:32}]; var result = arr.map(function (item) { item.age+=10; return item }); console.log(result);

4.filter

5.reduce
返回值是经过叠加处理后的操作。 函数中第一个参数为prev 第二个参数为next 求出价格中的总和

var arr = [{price:1},{price:2},{price:3}]; var result = arr.reduce(function (prev,next) { return prev+next.price; },0); console.log(result);

6.forEach
遍历数组 缺点:不能return循环

[1,2,3,4,5].forEach(function (item) { if(item==2) return; });

7.for in
缺点 1.会遍历所有属性 2.遍历的key是对象数据类型

var ary = [1,2,3]; arr.name = ‘jw’; for(var key in arr){ console.log(typeof key); console.log(arr[key]); }

8.for of
好处:可以break,不会遍历数组以外的属性 缺点:不能遍历普通对象
var ary = [1,2,3]; ary.name = ‘jw’; for(var value of ary){ if(key == 2){break} console.log(ary); }
